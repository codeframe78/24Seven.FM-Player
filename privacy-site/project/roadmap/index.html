M17 · Deferred

Private Messages are not in the shipping Alpha

Inbox and Sent Box discovery succeeded, but New Message selection remained suspect and a profile-originated MorgHubby test appeared to submit without delivery. The site owner has the reproduced result.

Development resumes only after the legacy server path is repaired and its production limits can be verified. The existing app does not pretend this capability works.

M25 · Planned after M24

Chat Mention Notifications

Add opt-in notifications when the signed-in member's exact station display identity is mentioned in Chat. The experience will be station-scoped, filter locally blocked authors, suppress duplicates, expose per-station controls and a dedicated notification channel, and deep-link through the existing community-safety gates.

Discovery gate: the current app has no developer backend and reads Chat only while the screen is selected. Reliable push requires an authorized station event source, webhook, or privacy-compatible relay. Continuous background polling will not be presented as push.

  • Authorized event delivery
  • Minimal payloads
  • No session forwarding
  • Privacy and Data Safety update
  • Battery and traffic validation
