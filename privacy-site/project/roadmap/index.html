M27 · Delivery boundary

One safe notification foundation

M27.1 now detects exact signed-in-name mentions from actively observed Chat, baselines existing messages, ignores blocked authors, suppresses duplicates, posts privacy-minimized Android alerts, and returns to gated station Chat. New Private Message notifications will reuse this contract only after M17 is repaired and verified.

M27.2 delivery gate: the app has no developer-operated backend and currently observes Chat only while its screen is selected. Reliable closed-app delivery requires an authorized station event source or privacy-compatible relay. Continuous background polling will not be presented as push.

  • Authorized true push
  • Minimal payloads
  • Separate Chat and PM controls
  • No session forwarding
  • Privacy and Data Safety review
  • Battery and traffic validation
